How to Use Goal Tracker

Getting Started

Goal Tracker helps you track your learning goals and daily tasks. Here's how to use the application:

  1. Create a new goal with a title, description, start date, and end date
  2. The system will automatically generate daily tasks for your learning journey
  3. Complete tasks daily to build a learning streak
  4. Track your progress on the dashboard

Dashboard

The dashboard shows your:

  • Current Streak: Number of consecutive days with completed tasks
  • Active Days: Total number of days with completed tasks
  • Missing Days: Number of days with incomplete tasks
  • Calendar: Visual representation of your progress
  • Goals and Tasks: List of all your goals and associated tasks

Completing Tasks

To complete a task:

  1. Click on today's date in the calendar or find today's task in your goals list
  2. Check the checkbox next to the task
  3. A validation dialog will appear where you must explain what you've learned
  4. The AI will evaluate your understanding and mark the task as complete if successful

You can only complete tasks for the current day. This ensures consistent learning practices.

Editing Tasks

You can edit task descriptions by clicking the edit button next to any task. Make your changes and click the save button to update the task.

Concept Chat

Use the Concept Chat feature to discuss learning concepts with the AI assistant. This can help you understand difficult topics or get additional explanations.

Tips for Success

  • Complete tasks daily to maintain your streak
  • Be specific when describing what you've learned during task validation
  • Use the calendar to plan ahead and track your progress visually
  • Break down complex learning goals into smaller, manageable pieces
